# Companion Robot

Companion Robot is a **robot implemented on Arduino** that interacts with users and provides **basic companionship through movement, sensors, and simple responses**.

## Features

- Human interaction using sensors  
- Basic movement and gestures  
- Voice or sound response (optional)  
- Obstacle detection and avoidance  
- Runs on Arduino  

## Hardware Required

- Arduino Uno  
- Ultrasonic sensor (HC-SR04)
- IR Sensor
- Servo motors / DC motors  
- Motor driver (L298N / L293D)
- BO Motor  
- Robot chassis  
- Battery pack  
- Jumper wires  

## Libraries Used

- Arduino built-in functions  
- Servo.h (for controlling servo motors)  

## File

- `companionbot_code.ino` – Main Arduino code for the companion robot.  

## How to Run

1. Connect the sensors, motors, and modules to the Arduino.  
2. Assemble the robot chassis with motors or servos.  
3. Upload `companionbot_code.ino` using the Arduino IDE.  
4. Power the robot using a battery pack.  
5. The robot will interact with users through movement, sound, or simple responses.
